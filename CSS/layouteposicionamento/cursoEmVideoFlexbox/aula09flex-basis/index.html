O flex-basis define o tamanho inicial de um item antes de o espaço extra ser distribuído pelo container flex. Ele pode ser em pixels, porcentagem ou 'auto'.

flex-basis: 50px → o item começa com 50 pixels de largura

flex-basis: 100px → o item começa com 100 pixels de largura

flex-basis: 150px → o item começa com 150 pixels de largura

flex-basis: 25% → o item ocupa 25% do container

flex-basis: 50% → o item ocupa 50% do container

flex-basis: auto → o tamanho do item é definido pelo conteúdo ou por width
