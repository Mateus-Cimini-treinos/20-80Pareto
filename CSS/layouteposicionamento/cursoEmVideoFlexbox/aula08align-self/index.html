O align-self permite que cada item dentro de um container flex seja alinhado individualmente no eixo cruzado, sobrescrevendo o align-items do container.

align-self: auto → herda o align-items do container

align-self: flex-start → o item fica no início do eixo cruzado

align-self: flex-end → o item fica no final do eixo cruzado

align-self: center → o item fica centralizado no eixo cruzado

align-self: baseline → o item se alinha à linha de base do texto

align-self: stretch → o item ocupa todo o espaço disponível no eixo cruzado
