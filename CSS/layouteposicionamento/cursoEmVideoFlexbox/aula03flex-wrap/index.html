O Flex-wrap serve para controlar se o o conteudo vai encapsula ou nao qunado o tamanho do container nao couber os itens filhos, por padrão vem configurado o no-wrap

Se o nowrap estiver ativo os itens vao diminuir de tamanho para caber dentro do container

Se o wrap estiver ativo os itens vão se manter do mesmo tamanho mas vai quebrar o layout no sentido do 'cross-axis'

Se o wrap-reverse estiver ativo o item vai se manter do mesmo tamanho mas vai quebrar o layout no sentido contrario ao 'cross-axis'

E tem o flew-flow que é a junção do Flex-wrap e flew-direvrion
