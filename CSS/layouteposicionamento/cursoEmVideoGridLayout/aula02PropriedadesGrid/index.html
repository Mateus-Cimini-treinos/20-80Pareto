Propriedades do Grid Container

Vamos analisar as principais propriedades usadas para criar um layout em CSS Grid. O exemplo a seguir é de um container com 7 itens, organizado em colunas e linhas.

Código usado: 

          div.container {
          display: grid;
          grid-template-columns: auto auto auto;
          grid-template-rows: 100px 100px 100px;
          grid-gap: 20px;
          }

display: grid;

Essa propriedade transforma a div em um Grid Container. A partir desse momento, todos os elementos filhos se tornam Grid Items, que podem ser organizados em linhas e colunas.

grid-template-columns: auto auto auto;

Define que o grid terá 3 colunas. Cada auto faz com que a largura da coluna seja ajustada automaticamente de acordo com o conteúdo.

grid-template-rows: 100px 100px 100px;

Define que o grid terá 3 linhas, cada uma com altura fixa de 100px. Isso organiza os itens de cima para baixo em faixas de tamanho igual.

grid-gap: 20px;

Define o espaçamento entre linhas e colunas do grid. Nesse caso, cada item tem um espaço de 20px entre si, deixando o layout mais organizado.
